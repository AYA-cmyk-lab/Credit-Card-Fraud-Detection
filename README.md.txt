 💳 Credit Card Fraud Detection

A machine learning project detecting fraudulent transactions with 95%+ accuracy.

🛠️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn, XGBoost
- Matplotlib, Seaborn

 🚀 How to Run
1. Install requirements:
```bash
pip install -r requirements.txt
```

2. Run the notebook:
```bash
jupyter notebook fraud_detection.ipynb
```
